# GeoJSON LineString

## Properties

- **`type`** *(string, required)*: Must be one of: `["LineString"]`.
- **`coordinates`** *(array, required)*: Length must be at least 2.
  - **Items** *(array)*: Length must be at least 2.
    - **Items** *(number)*
- **`bbox`** *(array)*: Length must be at least 4.
  - **Items** *(number)*
